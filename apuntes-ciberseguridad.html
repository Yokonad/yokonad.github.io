Otros conceptos: No repudio, spoofing, entropía, EOF en archivos

Archivos políglotas y acortadores con tracking (usar CheckShortURL para análisis)

Sniffing y ARP poisoning

Idea: interceptar tráfico en LAN mediante ARP spoofing para pruebas de MITM

dan@kali:~$ arpspoof -i eth0 -t 192.168.1.10 192.168.1.1
dan@kali:~$ bettercap -iface eth0
dan@kali:~$ ettercap -T -M arp:remote /192.168.1.10// /192.168.1.1//

ADVERTENCIA: actividad intrusiva — usar en entornos de prueba/autorizados

RFID / NFC y frecuencias

Frecuencias comunes: 125 kHz (LF) y 13.56 MHz (HF / NFC)

Hardware recomendado: Proxmark3, ChameleonMini, Flipper Zero

Casos de uso: análisis de control de accesos, clonación/emulación en laboratorios

Ética: no usar en tarjetas reales sin permiso

Seguridad Wi-Fi

Conceptos: beacon frames, modo monitor, evil-twin, deauth/jamming, captive portals

Vulnerabilidades: WEP (obsoleto), WPS (Pixie Dust), WPA/WPA2 (handshake, PMKID)

# Poner interfaz en modo monitor
dan@kali:~$ airmon-ng start wlan0

# Capturar handshake
dan@kali:~$ airodump-ng wlan0mon -c 6 --bssid AA:BB:CC:DD:EE:FF -w capture

# Deauth para forzar handshake
dan@kali:~$ aireplay-ng -0 10 -a AA:BB:CC:DD:EE:FF wlan0mon

# Crack con hashcat
dan@kali:~$ hashcat -m 2500 capture.hccapx wordlist.txt

Recordatorio legal: ataques a redes ajenas sin permiso son ilegales

SDR (Software Defined Radio)

Permite sintonizar/analizar un amplio espectro (AM/FM, ADS-B, GSM, IoT)

Hardware: RTL-SDR v3 (receptor barato), HackRF One (TX/RX), BladeRF/LimeSDR (profesional)

Software: rtl_433, GQRX, SDR++, pyModeS

dan@kali:~$ rtl_433 -f 433920000  # Decodificar sensores 433 MHz
dan@kali:~$ dump1090 --interactive  # ADS-B para tráfico aéreo

Criptología y esteganografía

Criptografía

Tipos: clásica (César/Vigenère), simétrica (AES), asimétrica (RSA, ECC)

Herramientas: OpenSSL, hashcat, John the Ripper, Hydra, hashid, CyberChef

dan@kali:~$ hashcat -m 1000 hashes.txt rockyou.txt  # NTLM
dan@kali:~$ john --wordlist=rockyou.txt hashes.txt
dan@kali:~$ hashid hash_example  # Identificar tipo de hash

Esteganografía

Técnicas: LSB en imágenes, EoF, metadatos, archivos políglotas

Herramientas: steghide, zsteg, binwalk, stegsolve, StegoVeritas, exiftool

dan@kali:~$ steghide extract -sf imagen.jpg
dan@kali:~$ zsteg imagen.png
dan@kali:~$ binwalk -e archivo_sospechoso
dan@kali:~$ exiftool imagen.jpg

Hacking web

Herramientas clave: Burp Suite, OWASP ZAP, curl, gobuster, Nikto, sqlmap, wfuzz

# Fuzzing de directorios
dan@kali:~$ gobuster dir -u http://target.com -w /usr/share/wordlists/dirb/common.txt

# SQLi automático
dan@kali:~$ sqlmap -u "http://target.com/page.php?id=1" --dbs

# Fuzzing de parámetros
dan@kali:~$ wfuzz -c -z file,wordlist.txt http://target.com/FUZZ

Vulnerabilidades: XSS, LFI/RFI, SSRF, SQLi, RCE, XXE

Metodología de pentest

Fases: Reconocimiento (OSINT) → Escaneo/Enumeración → Explotación/Escalada → Persistencia/Limpieza/Reporte

# OSINT
dan@kali:~$ whois target.com
dan@kali:~$ theHarvester -d target.com -l 500 -b google

# Escaneo
dan@kali:~$ nmap -sC -sV -oA scan target.com
dan@kali:~$ rustscan -a 192.168.1.0/24
